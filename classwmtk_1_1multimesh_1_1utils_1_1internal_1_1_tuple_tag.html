Detailed Description

TupleTag is a util helper class for tagging facets in a triangle mesh.

These tags can then be used to extract facet meshes as child mesh for multimesh.

The tagging algorithm is based on the union-find algorithm. It first tags all vertices by grouping them into disjoint sets. Then it creates facet tags based on the boundary tags. Vertex tag and facet tags are stored as mesh attributes of the parent triangle mesh, and can be accessed through "boundary_tag_handle" and "facet_tag_handle" respectively.
